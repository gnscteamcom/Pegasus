Configuração

Bem-vindo. Antes de começar, a aplicação tem de ser configurada para funcionar corretamente junto ao seu servidor. Você terá as seguintes informações durante instalação (se você não sabe esses itens, você pode obtê-los a partir de sua empresa de hospedagem):

1. Host do Banco de Dados
2. Porta do Banco de Dados (somente se diferente do padrão)
3. Nome do Banco de Dados
4. Usuário do Banco de Dados
5. Senha do Banco de Dados

Se o banco de dados ainda não existe, o assistente de instalação irá tentar criá-lo automaticamente para você. Em alguns servidores, no entanto, essa tentativa pode falhar. Mas não se preocupe, você pode criar o banco de dados manualmente usando sua ferramenta favorita de administração de banco de dados (e. g. phpMyAdmin ) e voltar a esta página depois de criado.
