Desintalação

Esta é a página de desinstalação. Se você não quiser mais usar o aplicativo, continue clicando no botão Uninstall.

Desinstalar significa que as contas das operadoras, história mensagens e mensagens predefinidas serão removidos. A conta de administrador, no entanto, permanecerá disponível e você pode entrar novamente mais tarde. tabelas de banco de dados serão removidos.

Após o processo de desinstalação for concluída, todos os dados do aplicativo (contas de operador, história, mensagens enlatadas) serão perdidos. Se necessário, antes de desinstalar, é possível tabelas de banco de dados aplicativo de backup usando sua ferramenta de administração de banco de dados favorito (e. G. phpMyAdmin ).
